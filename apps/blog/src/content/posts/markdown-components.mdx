# Custom MDX Components

Explainer v2 ships with several custom MDX components that are auto-injected into your content.

## Callouts

Use callouts to highlight important information:

<Callout variant="warning" title="Important">
  Always back up your data before upgrading.
</Callout>

<Callout variant="success" title="Done">
  Your configuration is complete!
</Callout>

## Tabs

Perfect for showing multiple alternatives:

<Tabs items={["React", "Vue", "Svelte"]} client:load>
  <Tab label="React">
    ```tsx
    export function App() {
      return <h1>Hello React</h1>
    }
    ```
  </Tab>
  <Tab label="Vue">
    ```vue
    <template>
      <h1>Hello Vue</h1>
    </template>
    ```
  </Tab>
  <Tab label="Svelte">
    ```svelte
    <h1>Hello Svelte</h1>
    ```
  </Tab>
</Tabs>

## Steps

Great for tutorials and guides:

<Steps>
  <Step title="Create a file">
    Create a new `.mdx` file in your content directory.
  </Step>
  <Step title="Add frontmatter">
    Add the required frontmatter fields like `title` and `description`.
  </Step>
  <Step title="Write your content">
    Use Markdown and the custom components to write your content.
  </Step>
</Steps>

All these components are available without any imports in your MDX files.
