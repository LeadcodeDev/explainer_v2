# Demarrage rapide

Bienvenue dans la documentation de **my-lib**. Ce guide vous aidera a demarrer rapidement.

## Installation

```bash
pnpm add my-lib
```

## Demarrage

Creez un nouveau fichier et importez la bibliotheque :

```typescript
import { createApp } from 'my-lib'

const app = createApp({
  name: 'Mon Application',
})

app.start()
```
