# Getting Started

Welcome to **my-lib** documentation. This guide will help you get up and running quickly.

## Installation

```bash
pnpm add my-lib
```

## Quick Start

Create a new file and import the library:

```typescript
import { createApp } from 'my-lib'

const app = createApp({
  name: 'My Application',
})

app.start()
```
