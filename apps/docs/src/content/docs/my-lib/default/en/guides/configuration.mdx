# Configuration

Learn how to configure my-lib for your specific needs.

## Configuration File

Create a `my-lib.config.ts` at the root of your project:

```typescript
import { defineConfig } from 'my-lib'

export default defineConfig({
  name: 'My Application',
  port: 3000,
  plugins: [],
})
```

## Options

### `name`

The name of your application. Used in logs and error messages.

### `port`

The port your application will listen on. Defaults to `3000`.

### `plugins`

An array of plugins to extend functionality.

<Callout variant="warning" title="Plugin Order">
  Plugins are executed in the order they are defined. Make sure to place middleware plugins before route plugins.
</Callout>

## Environment Variables

You can override configuration with environment variables:

```bash
MY_LIB_PORT=8080
MY_LIB_NAME="Production App"
```
