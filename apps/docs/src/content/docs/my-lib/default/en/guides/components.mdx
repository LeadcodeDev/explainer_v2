### Card item

You can customize each card using the following props:

- `label`: The text to display at the top of the card. (required)
- `icon`: The icon to display at the top of the card.

:::callout{variant="info"}
The cards use the [`iconify`](https://icon-sets.iconify.design/) library to display icons.

## Code Blocks

Code blocks are a way to display code in a readable format.

### Highlighting

#### Single line

Single highlight can be done with the `[!code highlight]` syntax.

::::preview
:::code-preview

```rs
pub struct User {
  // [!code highlight]
  pub firstname: String,
  pub lastname: String,
}
```

:::

:::content-preview

````mdx
```
pub struct User {
  // [!code highlight]
  pub firstname: String,
  pub lastname: String,
}
```
````

:::
::::

#### Multiple lines

Multiple highlights can be done with the `{1,3-4}` syntax.

::::preview
  :::code-preview
  ```rust {1,3-4}
  pub struct User {
    pub firstname: String,
    pub lastname: String,
  }
  ```
  :::
  :::content-preview
  ````mdx
  ```rust {1,3-4}

    pub firstname: String,
    pub lastname: String,
  }
  ```
  ````

### Line modifiers

The `[!code --]` and `[!code ++]` syntax will remove or add a line to the code block.

::::preview
  :::code-preview
  ```rust
  pub struct User {
    pub firstname: String,
    pub lastname: String,
  }
  
  impl User {
    pub fn new() -> Self { // [!code --]
    pub fn new(firstname: String, lastname: String) -> Self { // [!code ++]
      Self { firstname, lastname }
    }
  }
  ```
  :::

  :::content-preview
  ````mdx
    ```
    pub struct User {
      pub firstname: String,
      pub lastname: String,
    }
    
    impl User {
      pub fn new() -> Self { // [!code --]
      pub fn new(firstname: String, lastname: String) -> Self { // [!code ++]
        Self { firstname, lastname }
      }
    }
    ```
  ````

### Group code blocks

:::callout{variant="warning"}
Only code blocks can be used in a group.
:::

::::code-preview

```rs [foo]
  pub firstname: String,
  pub lastname: String,
}
```

```rs [bar]

  pub firstname: String,
  pub lastname: String,
```
