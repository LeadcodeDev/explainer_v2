# Installation

<Callout variant="info" title="Prerequisites">
  You need Node.js 20+ and pnpm installed on your machine.
</Callout>

## Package Managers

<Tabs items={["pnpm", "npm", "yarn"]} client:load>
  <Tab label="pnpm">
    ```bash
    pnpm add my-lib
    ```
  </Tab>
  <Tab label="npm">
    ```bash
    npm install my-lib
    ```
  </Tab>
  <Tab label="yarn">
    ```bash
    yarn add my-lib
    ```
  </Tab>
</Tabs>

## Setup Steps

<Steps>
  <Step title="Install the package">
    Run the install command above with your preferred package manager.
  </Step>
  <Step title="Create configuration file">
    Create a `my-lib.config.ts` file at the root of your project.
  </Step>
  <Step title="Start developing">
    Run `pnpm dev` to start the development server.
  </Step>
</Steps>
