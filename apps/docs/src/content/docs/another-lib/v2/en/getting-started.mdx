# Getting Started with another-lib v2

Welcome to **another-lib** v2 documentation. This version includes breaking changes.

## Installation

```bash
pnpm add another-lib@2
```

## Migration from v1

<Callout variant="warning" title="Breaking Changes">
  v2 uses a new configuration format. See the migration guide.
</Callout>

## Usage

```typescript
import { createInstance } from 'another-lib'

const instance = createInstance({
  mode: 'production',
  features: ['cache', 'logging'],
})
```
